🛰️ Satellite Scene Automated Classifier

An end-to-end deep learning application featuring a fine-tuned **ResNet18** convolutional neural network designed to accurately classify remote sensing and natural scene images into six distinct geographic categories. Built using **PyTorch** and deployed via an interactive **Streamlit** user interface.

## 📊 Performance Summary
* **Base Architecture**: ResNet18 (Pre-trained on ImageNet)
* **Macro Average F1-Score**: 86%
* **Target Classes**: Buildings, Forest, Glacier, Mountain, Sea, Street
